    Welcome to Persistentlayer.nhibernate.

    PersistentLayer.NHibernate makes easier architecture development based on ORM nhibernate, which can be used in any kind of context whenever It requires to make CRUD operations on data stores.

    Context of use

    This framework can be use easily into any kind of context, example:

    • Windows application
    • WCF
    • ASP.NET ( Web forms / MVC / WEB API )
    • ESB ( Enterprise Service Bus )

    Knowledge about Nhibernate?

    In order to understand better this framework, It's required to know minimal about Nhibernate, mainly query providers (ex. HQL, DetachedCriteria, QueryOver, and the famous LINQ syntax) in order to take full advantage of this framework.
