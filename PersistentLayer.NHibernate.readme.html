Welcome to PersistentLayer.NHibernate

PersistentLayer.NHibernate is an easy component used by retrieving application data which its implementations is based on NHibernate.

Latest Version

The quickest way to get the latest release of PersistentLayer.NHibernate is to add it to your project using NuGet (http://nuget.org/List/Packages/PersistentLayer.NHibernate).
