Release notes:

New interfaces for new DAO hierarchies:
- interface IRootQueryableDAO<in TRootEntity>
- interface IRootQueryableDAO<in TRootEntity, TEntity>
- interface IRootPersisterDAO<in TRootEntity>
- interface IRootPersisterDAO<in TRootEntity, TEntity>
- interface IRootPagedDAO<in TRootEntity>
- interface IRootPagedDAO<in TRootEntity, TEntity>

With these new interfaces, you can apply DDD pattern in order to apply "Root entities" which can be defined as Aggregate Entity types for DAOs.


That's All.
